# WinDevCamp
Windows 10 Developer Training (XAML / C#)

December 2015 Release (Build 10586)

This session was a part of the half-day developer training for Windows 10 Version 1511 now available on-demand on Channel 9, [Developer’s Guide to Windows 10 Version 1511] (https://channel9.msdn.com/Events/Windows/Developers-Guide-to-Windows-10-Version-1511), by Andy Wigley and Shen Chauhan. 

[Video: What's New for Developers in Windows 10 Version 1511 and the 10586 SDK](https://channel9.msdn.com/Events/Windows/Developers-Guide-to-Windows-10-Version-1511/Whats-New-for-Developers-in-Windows-10-Version-1511-and-the-10586-SDK)

Demos used in this module include the JumpList and Bluetooth pairing demos found in the official [UWP demos on Github](https://github.com/Microsoft/Windows-universal-samples/)
