# WinDevCamp
Windows 10 Developer Training (XAML / C#)

December 2015 Release (Build 10586)

This folder contains the content for the half-day developer training for Windows 10 Version 1511 now available on-demand on Channel 9, [Developer’s Guide to Windows 10 Version 1511] (https://channel9.msdn.com/Events/Windows/Developers-Guide-to-Windows-10-Version-1511). This content is a supplement to the original Build 10240 developer training that you can find in the [Presentation](https://github.com/Windows-Readiness/WinDevCamp/tree/master/Presentation) folder.
